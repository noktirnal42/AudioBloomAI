AudioBloomAI

Next-generation audio visualizer optimized for Apple Silicon

Experience stunning real-time audio visualization powered by Neural Engine integration and a Metal-based graphics pipeline.

Explore Features View on GitHub

Key Features

Neural Engine Integration

Harness the power of Apple's Neural Engine for sophisticated audio analysis and pattern recognition, providing unique visualization capabilities.

Metal-Based Graphics

Leveraging Apple's Metal framework for blazing-fast graphics rendering, delivering smooth, high-performance visualizations with minimal CPU overhead.

Apple Silicon Optimization

Specifically optimized for Apple Silicon M3 Pro, taking full advantage of the unified memory architecture and performance cores.

Real-time Audio Processing

High-fidelity audio capture and processing with minimal latency, perfect for live performances and audio production environments.

System Requirements

  • Development Environment Xcode 16+
  • Language Version Swift 6
  • Recommended Hardware Apple Silicon (M-series)

Installation Guide

Clone the Repository
git clone https://github.com/noktirnal42/AudioBloomAI.git
cd AudioBloomAI
Open in Xcode

Open the project in Xcode 16 or later:

open Package.swift
Build and Run

Select your target device and click the Run button in Xcode, or use the keyboard shortcut ⌘+R.

Note: For detailed setup instructions, refer to the Development Setup Guide.
